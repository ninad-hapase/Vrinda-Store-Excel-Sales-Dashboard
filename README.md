# Excel Sales Dashboard – Data Analytics Project

## 📌 Project Overview

This project is an interactive Excel Sales Dashboard created to analyze sales performance, customer behavior, and order trends using Microsoft Excel.

The dashboard transforms raw sales data into meaningful business insights through data cleaning, processing, pivot tables, charts, and interactive visualizations.

The objective of this project is to help businesses monitor sales performance, identify trends, and support data-driven decision-making.

## 📊 Dashboard Features

* Interactive Sales Dashboard
* Order and Revenue Analysis
* Customer Demographics Insights
* Sales Trends Visualization
* Category and Channel Analysis
* KPI Cards and Charts
* Dynamic Filters and Slicers
* User-Friendly Dashboard Design
